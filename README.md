## Proyecto de Change Request Forms

Formulario sencillo para pedir las propuestas de cambios. Usa NodeJS, JS, HTML, CSS y MySQL

## Comandos
**Primera vez**
<br>
npm install
<br>

**Para que corra el servidor**
<br>
npm run dev
<br>

**Para hacer las migraciones**
<br>
npm run migrate
